# R-Script-Portfolio
A collection of data analysis and economic modeling projects, primarily utilizing R script, including data cleaning, visualization and statistical methods.

Project #1 - Basic Titanic Dataset Analysis
This project performs an exploratory data analysis of the Titanic dataset using R to identify patterns in passenger survival. The dataset is transformed from aggregated counts into passenger-level observations, followed by data visualization and statistical modeling. A logistic regression model is implemented to predict survival based on passenger class, gender, and age group. The model is evaluated using a train–test split and a confusion matrix, demonstrating practical applications of data preprocessing, statistical analysis, and predictive modeling in R.

Project #2 - Interactive Titanic Survival Explorer (Shiny App) 
This project develops an interactive Shiny application that allows users to explore Titanic survival data dynamically. The app enables filtering by passenger class, gender, and age group, and updates results in real time to display survival rates, filtered datasets, and visualizations. Built using Shiny, dplyr, ggplot2, and plotly, the application demonstrates how data analysis can be transformed into an interactive dashboard for exploring patterns and insights within a dataset.
